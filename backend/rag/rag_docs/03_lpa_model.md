# LPA 潜在剖面分析模型

## 模型原理
LPA（Latent Profile Analysis）是一种以个体为中心的分类方法。
本系统基于各省 2016-2024 年的综合得分轨迹，将 31 省分为 4 种
发展类型，识别不同省份的绿色算力承载能力演化模式。

## 四种类型

### 类型 1：高位领先型（5 省）
- 特征：综合得分最高，持续领先
- 典型省份：北京、上海、江苏、广东、浙江

### 类型 2：优势支撑型（4 省）
- 特征：得分较高，有特定优势领域支撑
- 典型省份：天津、山东、福建、湖北

### 类型 3：中位追赶型（9 省）
- 特征：中等得分水平，增速较快
- 典型省份：河北、山西、内蒙古、辽宁、安徽、江西、河南、湖南、四川

### 类型 4：基础培育型（13 省）
- 特征：得分较低，处于基础培育阶段
- 典型省份：吉林、黑龙江、广西、海南、重庆、贵州、云南、西藏、陕西、甘肃、青海、宁夏、新疆

## 模型选择依据
通过比较 2-4 类的 AIC、BIC、Entropy 指标，选择 4 类方案（Entropy 最高，分类最清晰）。
