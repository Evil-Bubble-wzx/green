# 障碍度诊断模型

## 模型原理
障碍度模型用于识别制约各省绿色算力承载能力提升的关键障碍因素。
通过计算各维度/指标的障碍度，确定首要和次要障碍维度。

## 障碍度计算公式
障碍度 = (指标权重 × 目标差距) / Σ(指标权重 × 目标差距) × 100%

其中目标差距 = 1 - 标准化值（标准化值越接近 1 表示越接近最优水平）

## 诊断层次
1. **全国维度障碍度** — 全国层面，哪个维度是最大短板
2. **区域维度障碍度** — 东部/中部/西部/东北各自的障碍结构
3. **LPA 类型障碍度** — 四种发展类型的障碍差异
4. **省份短板诊断** — 每省的首要/次要障碍维度与短板类型
5. **二级指标障碍度** — 34 个指标的障碍度排名

## 短板诊断类型
- 创新需求协同型：创新短板 + 需求不足
- 能源低碳约束型：能源供给不足 + 低碳压力大
- 基础数字设施型：数字基础设施薄弱
- 综合约束型：多方面同时受限
