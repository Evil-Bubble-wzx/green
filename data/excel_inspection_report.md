## Field Ambiguity Notes

The following fields MAY need manual confirmation:

1. **Province names** — Verify all province names match the standard short form
   (e.g., "内蒙古" not "内蒙古自治区").
2. **Year values** — Confirm all year values fall within 2016-2024 range.
3. **Numeric fields** — Check for unexpected string values in numeric columns.
4. **Unnamed columns** — Some sheets (e.g., Markov transition matrices) have
   unnamed first columns. Column mapping in the import config must handle these.
5. **Region values** — Verify region values are one of: 东部, 中部, 西部, 东北.
6. **LPA types** — Verify LPA type values match the 4 defined types.
